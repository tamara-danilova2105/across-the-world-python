## Восстановление зависимостей (если забыли, что установлено)

Если проект был создан давно и список зависимостей потерян, его можно восстановить из существующего виртуального окружения.

### Важно
Команда должна выполняться **только при активированном virtualenv**.

В терминале слева должно быть видно `(venv)`.

### Шаги

1. Активировать виртуальное окружение

Windows (PowerShell):
    venv\Scripts\Activate.ps1

macOS / Linux:
    source venv/bin/activate

2. Сгенерировать файл `requirements.txt` из текущего окружения

    python -m pip freeze > requirements.txt

3. Проверить содержимое файла

    type requirements.txt        # Windows
    cat requirements.txt         # macOS / Linux

В файл будут записаны все установленные зависимости и их версии.

### Примечание
Если выполнить `pip freeze` **до активации venv**, файл может оказаться пустым или содержать зависимости из глобального Python.
